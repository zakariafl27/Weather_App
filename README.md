# Weather App

Modern weather application built with React.js featuring interactive D3.js charts, animated landscape illustrations, and real-time weather data.

## Features

- Search from 50+ cities worldwide with autocomplete
- Interactive D3.js charts for temperature, precipitation, and wind
- Animated landscape with day/night mode
- 7-day weather forecast
- Real-time weather data from OpenWeatherMap API
- Fully responsive design
- Modern UI with glassmorphism effects

## Installation

```bash
# Clone repository
git clone https://github.com/zakariafl27/Weather_App.git

# Navigate to directory
cd Weather_App

# Install dependencies
npm install

# Create .env file and add your API key
echo "REACT_APP_WEATHER_API_KEY=your_api_key_here" > .env

# Start application
npm start
```

Get your free API key at [openweathermap.org](https://openweathermap.org/api)

## Technologies

- React.js 18.2.0
- D3.js 7.8.5
- Tailwind CSS
- Lucide React
- OpenWeatherMap API
